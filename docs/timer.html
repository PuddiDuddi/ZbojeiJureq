# Timer Timer plugin lets you send messages into the future. ## Commands ### .timer [xx.xx] [d|h|m|s] message leaves a message that will be sent in xx.xx days/hours/minutes/seconds ######Examples: - *.timer 15.50 m hello* will send the message *hello* after 15minutes and 30 seconds - *.timer 20.50 d hello* will send the message *hello* after 20days and 12 hours ### .timer [hh:mm] message leaves a message that will be sent at hh:mm, where hh-hours, mm-minutes Examples: - *.timer 15:50 hello* will send the message *hello* at 15:50 - *.timer 15:50 hello* **when it's 16:30** will send the message *hello* at 15:50 the following day. ### .timer [dd.mm.yyyy] [hh:mm] message leaves a message on the date dd.mm.yyyy at hh:mm. Examples: - *.timer 30.12.2020 23:20 hello* will send the message *hello* at 23:20 on 30.12.2020
